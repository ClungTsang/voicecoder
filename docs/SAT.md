## 1. 验收总则

### 1.1 验收流程

```
PRD 功能项 → 验收用例 → 执行测试 → 记录结果 → Pass/Fail
                                          ↓
                              Fail → 修复 → 回归测试 → Pass
```

### 1.2 验收标准

- **P0 必做项**：100% Pass 才能发布
- **P1 可选项**：≥ 80% Pass 可发布
- **P2 远期项**：不纳入本期验收

### 1.3 测试环境

| 环境 | 要求 |
|------|------|
| macOS | macOS 14+ Apple Silicon（M1/M2/M3）+ Intel Mac |
| Windows | Windows 10 x64 + Windows 11 ARM64 |
| 后端 | voicecoder_service.py v4 已启动 |
| 网络 | 需联网（GitHub OAuth + 检查更新 + 模型下载） |
